A calculadora permite realizar as seguintes operações:

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- ⬆️ Potência
- ✔️ Raiz Quadrada

Conceitos utilizados

- Leitura de dados com `Scanner`
- Condicionais `if` e `else`
- Laços de repetição `while` ou `do-while`
- Operadores matemáticos
